Level 4: Predict daily accidents from hourly weather

Upload CSV

Upload hourly meteorological data (same structure as metherology_dataset.csv).

How It Works

  • We group 24 hourly rows by location and date.
  • Aggregate weather features (mean per day) and one-hot encode locations.
  • Run a trained XGBoost regressor to predict accidents per day.
  • Results are non-negative integers (rounded).
